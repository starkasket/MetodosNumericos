Bienvenidos

Proyecto de la materia de Métodos Numéricos para el desarrollo de los distintos algoritmos aplicados a la ingeniería utilizando:

  • Java JDK 21
  • SpringBoot 3.5.0
  • Lombok 1.18
  • Thymeleaf 3.1.13
  • MathParser 6.1.0

Soluciones implementadas

El alcance de este proyecto incluye:

  • Algoritmos de solución de ecuaciones para estimación de raíces
  • Algoritmos de solución sistemas lineales y no lineales
  • Algoritmos de ajuste de datos
  • Algoritmos de diferenciación e integración
  • Algoritmos para solución de ecuaciones diferenciales

Métodos Numéricos

Los métodos numéricos son procedimientos matemáticos que utilizan operaciones aritméticas y lógicas para obtener soluciones aproximadas a problemas complejos que no pueden resolverse mediante métodos analíticos. Estos métodos se basan en algoritmos, que son secuencias finitas de instrucciones ordenadas que permiten acercarse a una solución real. Su objetivo principal es encontrar soluciones aproximadas a problemas matemáticos, especialmente aquellos que son demasiado complejos para una resolución exacta.

En la práctica, los métodos numéricos se aplican ampliamente en ingeniería, física, química y otras ciencias para resolver problemas como ecuaciones diferenciales ordinarias, sistemas de ecuaciones lineales, ecuaciones no lineales y problemas de valor inicial o de contorno.
